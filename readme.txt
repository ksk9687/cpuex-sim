*アセンブラの使い方
起動方法は java asm.Assembler
入出力はリダイレクトで行う
-encoding hogeでエンコーディングを指定できる(デフォルトはUTF-8)

*アセンブリ言語の仕様
基本的に機械語の通りに命令とオペランドを空白区切りで並べて書く
レジスタの前には$を付ける
大文字と小文字は区別されない
raw nと書くとnが直接機械語になる (定数テーブルやグローバル変数などに使う)
#から後ろがコメントになる
ラベルはhoge: (:の前に空白は禁止)
jmpやloadなどの即値としてラベルを指定することができる
数値の指定方法は、10進(そのまま)、16進(0xを前に付ける)、2進(0bを前に付ける)に対応
rawの場合は小数も使える

*シミュレータの使い方
起動方法は java sim.Simulator バイナリファイル
入出力はリダイレクトで行う
バイナリ実行機能しかまだないので、ステップ実行やレジスタ＆メモリの中身が見たかったらEclipse上でデバッグ実行してください
